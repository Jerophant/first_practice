rel="noopener noreferrer"

This is an HTML link attribute used to enhance security and privacy when linking to external websites, particularly when the link opens in a new tab or window using:


  target="_blank"

ATTRIBUTE

rel

  • indicates the relationship between the current page and the linked resource
  • used in <a>, <area> and <link>

TOKEN

noopener

  • link type that tells the browser not to run window.opener when a new page is opened
  • usually with target="_blank"
  • result: the opened page cannot manipulate the opening page (mitigating risks like reverse-tabnabbing)

noreferrer

  • link type that tells the browser not to send the Referer header to the destination site/page
  • result: the destination site does not see which url you are coming from
  • note: in modern browsers, noreferrer implicitly includes the effects of noopener, though they’re both written for ☆ clarity ☆

EXAMPLE


  <a href="https://example.com" target="_blank" rel="noopener noreferrer">Example Site</a>

HOW AND WHEN TO USE IT

External link + new window


  rel="noopener noreferrer"

If you need referrer for analytics


  rel="noopener"

OTHER TOKENS USED WITH rel

stylesheet, preload, icon, canonical, etc…
